## 💡 How This Project Uses These Resources

### Data Processing & Analysis
- **Pandas** for loading and preprocessing ATP match data
- **NumPy** for numerical operations
- **SciPy** for scientific computing

### Machine Learning
- **Scikit-learn** for all 4 models:
  - DecisionTreeClassifier (simple, interpretable)
  - MLPClassifier (neural networks)
  - RandomForestClassifier (ensemble, best performer)
  - KMeans (clustering for exploration)

### Model Evaluation
- **Scikit-learn metrics** for accuracy, precision, recall, F1, ROC-AUC
- **train_test_split** for 80/20 train-test split

### Data Preprocessing
- **ColumnTransformer** for parallel pipeline
- **StandardScaler** for numerical features
- **OneHotEncoder** for categorical features
- **LabelEncoder** for player names

### Testing
- **Python unittest** for 12 comprehensive tests
- **Pickle** for model serialization
